# CampusMarket

# Descrição
## Sistema Web / Mobile para comercialização de alimentos (lanches) entre os alunos no ambiente escolar.

# Objetivo
## Organizar vendas e compras informais dentro da instituição escolar estimulando:
- Organização
- Segurança
- Transparência
- Autonomia
- Empreendedorismo Jovem

# Impacto no Convívio Escolar
# O sistema pode:
- Reduzir possíveis conflitos entre alunos;
- Organizar vendas informais;
- Incentivar empreendedorismo;
- Desenvolver responsabilidade digital.
## Além disso, pode servir como ferramenta educativa prática de gestão de negócios.
 
# Usuários
- Vendedores
- Clientes

# Tecnologias / Metodologia
- Frontend (HTML, CSS e Bootstrap)
- Backend (Node, Javascript e Express)
- Banco de Dados (Mysql)

# Ferramentas
- Github Pages (Hospedagem do site)
- Railway (Hospedagem backend e Banco de Dados)

# Cronograma
- Planejamento "Estrutura e ideias definidas"
- Modelagem do Banco de Dados "Desenvolvimento do Banco de Dados"
- Backend "API + CRUD em desenvolvimento"
- Frontend "Desenvolvimento FrontEnd"
- Diferenciais "Sistema avançado"
- Deploy + Teste "Sistema em modo Beta"
- Apresentação "Conclusão"
